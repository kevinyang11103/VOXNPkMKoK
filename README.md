# 图书管理系统 java31

## 项目概述

图书管理系统是一个基于 Spring Boot + Vue 实现的前后端分离项目，旨在提供一套完整的图书管理解决方案。

## 技术栈

### 后端技术

- Spring Boot
- MyBatis-Plus

### 前端技术

- Vue
- Axios
- Element-UI

### 开发环境

- Java 1.8
- Node.js
- MySQL 8
- Maven

## 功能模块

### 管理员端

- 首页
- 系统管理
  - 用户管理
  - 菜单管理
  - 角色管理
- 读者管理
  - 读者列表
- 图书管理
  - 图书分类
  - 图书列表
- 借阅管理
  - 借阅管理
  - 还书管理
  - 借阅查看
- 公告管理
  - 公告列表

### 读者端

- 借阅管理
  - 借阅查看
- 公告管理
  - 公告列表
